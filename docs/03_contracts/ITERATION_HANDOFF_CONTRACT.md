# Iteration handoff contract

A handoff is complete only when the next contributor can answer all of the following quickly:

- What was the slice?
- What changed?
- What passed?
- What failed?
- What remains divergent?
- What is next?

## Mandatory handoff artifacts

- updated `trackers/NEXT_ITERATION.md`
- latest implementation log entry
- latest verification status update
- divergence entries for unresolved issues
